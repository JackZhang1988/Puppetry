# Puppetry
网页End2End测试GUI工具

#### 示例:  


### 特性
- 无需编写代码模拟浏览器行为
- 模拟脚本实时预览
- 支持基于image diff的css回归测试
- 自动检测page error、page performance

### 使用方法
安装Mongodb


安装node_module 
``` 
npm install
```
运行
``` 
npm run server
```
访问： http://localhost:3000/#/ 

### 注意事项
* 如果在Linux服务器安装，请保证服务器版本可以安装puppeteer, 如果是centos系统，保证系统版本为7.0以上, 参考：https://github.com/GoogleChrome/puppeteer/issues/497
